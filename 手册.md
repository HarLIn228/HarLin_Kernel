## 内核概述

HarLin Kernel 是由 HarLin228 Studio 开发的 x86_64 通用操作系统内核。本项目采用统一 API 层设计，第三方开发者可基于本内核修改实现、添加新功能并在遵守 MIT 开源协议的前提下发布衍生作品。

当前版本已移除内置 Shell，启动后直接进入静默等待状态，所有扩展功能由第三方开发者通过 API 实现。

## 内存布局

| 地址范围 | 用途 | 说明 |
|----------|------|------|
| 0x00000000 - 0x000003FF | 实模式中断向量表 | 保留 |
| 0x00000400 - 0x000004FF | BIOS 数据区 | 保留 |
| 0x00000500 - 0x00007BFF | 可用常规内存 | 约 30KB |
| 0x00007C00 - 0x00007DFF | 引导扇区 | 512 字节 |
| 0x00010000 - 0x0009FFFF | 内核加载区 | 约 520KB |
| 0x000A0000 - 0x000B7FFF | 图形帧缓冲 | VGA 图形模式 |
| 0x000B8000 - 0x000BFFFF | VGA 文本缓冲区 | 80x25 字符显示 |
| 0xFFFF800000000000 | 内核高半区起始 | 第三方可扩展映射 |

## 统一 API 接口

所有功能通过 harlin_API.h 中声明的接口访问，入口函数为 Harlin_Boot()。

### 系统控制

```c
void Harlin_Boot(void);
void Harlin_Shutdown(void);
```

- `Harlin_Boot()`：内核初始化入口，负责设置中断、键盘、网络、内存管理器并显示启动信息。
- `Harlin_Shutdown()`：停止系统，关闭中断并进入停机循环。

### 控制台输出

### 显示模式

```c
#define HARLIN_DISP_VGA_TEXT 0
#define HARLIN_DISP_VGA_13H  1
#define HARLIN_DISP_VESA     2

int  Harlin_DisplaySetMode(int mode);
void Harlin_DisplayClear(unsigned char color);
```

当前内置驱动仅支持 HARLIN_DISP_VGA_TEXT。HARLIN_DISP_VGA_13H 与 HARLIN_DISP_VESA 为第三方扩展点，需在 display.c 中接入对应驱动实现。

### 磁盘 I/O


当前支持 ATA PIO LBA28 模式访问主通道 IDE 硬盘。第三方可在此基础上实现 MBR/GPT 分区解析与文件系统。

### 分区表

```c
struct Harlin_PartitionInfo {
    u8  active;
    u8  type;
    u32 start_lba;
    u32 sector_count;
};

int Harlin_PartitionInit(void);
int Harlin_PartitionCount(void);
int Harlin_PartitionGet(int index, struct Harlin_PartitionInfo* out);
```

解析 LBA 0 处的 MBR，获取最多 4 个主分区信息。

### FAT32 文件系统

```c
struct Harlin_File {
    u32 start_cluster;
    u32 current_cluster;
    u32 position;
    u32 size;
};

#define HARLIN_FS_OK     0
#define HARLIN_FS_ERROR -1
#define HARLIN_FS_EOF   -2

int  Harlin_FsMount(u32 partition_lba);
int  Harlin_FsOpen(const char* name, struct Harlin_File* out);
int  Harlin_FsRead(struct Harlin_File* file, void* buf, u32 len);
u32  Harlin_FsSize(struct Harlin_File* file);
void Harlin_FsClose(struct Harlin_File* file);
```

当前为只读实现，支持短文件名，挂载 FAT32 分区后打开根目录下的文件并读取内容。

## 第三方开发指南

### 添加显示驱动

在 src/Sys_C/display.c 的 display_set_mode 函数中接入新驱动：

```c
if (mode == HARLIN_DISP_VGA_13H) {
    vga13_init();
    current_mode = mode;
    return 0;
}
```

并相应实现 display_clear / display_put_pixel / display_put_string 的分支逻辑。

### 添加硬件驱动

在 src/Sys_C/ 目录下创建新驱动源文件，在 harlin_API.c 的 Harlin_Boot() 中初始化，并通过 harlin_API.h 暴露接口。

### 添加内存分配器

可基于 Harlin_PmmAlloc 与 Harlin_PmmFree 实现 slab、buddy 等高级分配器，并通过 harlin_API.h 暴露 kmalloc/kfree 接口。

### 注意事项

- 内核运行于 x86_64 64位长模式，已启用分页与内存管理
- 无标准 C 库，使用内核提供的工具函数
- 所有 API 区分大小写
- 第三方扩展需遵守 MIT 开源协议
