## 内核二次开发

### 内核架构

| 属性 | 规格 |
|------|------|
| 架构 | x86 (i386) |
| 运行模式 | 32位保护模式 |
| 引导方式 | MBR 引导扇区 |
| 文件系统 | 无 (原始镜像) |
| 显示模式 | VGA Text 80x25 |
| 视频内存 | 0xB8000 |
| 输入设备 | PS/2 键盘 |

### 构建系统

#### 目录结构

```
HarLIn_Boot/
├── build/          构建输出目录
├── src/
│   ├── ASM/        汇编源文件
│   ├── Sys_C/      C源文件
│   └── head/       头文件
├── build.bat       Windows构建脚本
└── run.bat         运行脚本
```

#### 编译工具链

| 工具 | 用途 | 版本要求 |
|------|------|----------|
| NASM | 汇编编译 | 2.14+ |
| GCC | C编译 | MinGW-w64 i686 |
| LD | 链接 | GNU Binutils |

#### 编译命令

```bash
# Windows
build.bat

# 手动编译
nasm -f bin src/ASM/boot.asm -o build/boot.bin
gcc -ffreestanding -c -m32 -O2 -Wall -I src/head -o build/kernel.o src/Sys_C/kernel.c
ld -T src/Sys_C/linker.ld -o build/kernel.bin build/kernel.o -m i386pe -nostdlib
```

### 内存布局

| 地址范围 | 用途 | 说明 |
|----------|------|------|
| 0x00000000 - 0x000003FF | 实模式中断向量表 | 保留，不可使用 |
| 0x00000400 - 0x000004FF | BIOS 数据区 | 保留，不可使用 |
| 0x00000500 - 0x00007BFF | 可用常规内存 | 约 30KB 可用空间 |
| 0x00007C00 - 0x00007DFF | 引导扇区 | 512字节，内核入口 |
| 0x00007E00 - 0x0009FFFF | 内核加载区 | 扩展内存，约 600KB |
| 0x000A0000 - 0x000B7FFF | 显示内存 | VGA图形模式 |
| 0x000B8000 - 0x000BFFFF | VGA文本缓冲区 | 80x25字符显示 |

### I/O 端口映射

| 端口 | 设备 | 读/写 | 用途 |
|------|------|-------|------|
| 0x60 | 键盘控制器 | R | 键盘扫描码读取 |
| 0x64 | 键盘控制器 | R/W | 命令/状态寄存器 |
| 0x70 | CMOS/RTC | W | 寄存器索引选择 |
| 0x71 | CMOS/RTC | R/W | CMOS数据读写 |
| 0x3D4 | VGA CRTC | W | CRT控制器索引 |
| 0x3D5 | VGA CRTC | R/W | CRT控制器数据 |

### API 接口

#### 屏幕输出

```c
void screen_clear(void);           // 清屏
void screen_put_char(char c);      // 输出单个字符
void screen_put_string(const char* s);  // 输出字符串
```

#### 键盘输入

#### I/O 操作

```c

```

### 扩展开发

#### 添加新命令

在 shell.c 中添加命令处理函数：

```c
static void cmd_mycmd(const char* args)
{
    // 命令实现
}

// 在 exec_cmd() 中添加：
else if (cmd_match(p, "mycmd")) { cmd_mycmd(cmd_args(p)); }
```

#### 添加系统调用

内核目前无系统调用机制，直接调用内核函数即可。

#### 驱动开发

在 src/Sys_C/ 目录下创建新驱动文件，在 kernel.c 中初始化。

### 开发者注意事项

- 内核运行于 32 位保护模式，无虚拟内存
- 无中断处理，使用轮询方式处理输入
- 无堆内存管理，使用静态分配
- 标准 C 库不可用，使用内核提供的函数
- 所有命令区分大小写
- 输入缓冲区限制 256 字节
- 变量最大数量 64 个

## Shell 指令

### 内置命令

| 命令 | 语法 | 功能描述 |
|------|------|----------|
| help | help | 显示所有可用命令列表 |
| cls | cls | 清除屏幕内容，光标复位 |
| say | say <text> | 输出文本到屏幕 |
| ver | ver | 显示系统版本信息 |
| info | info | 显示系统运行模式信息 |
| reboot | reboot | 通过键盘控制器复位系统 |
| set | set <var> = <expr> | 定义或修改变量值 |
| show | show [var] | 显示变量值，省略参数则列出所有变量 |
| date | date | 从 CMOS 读取实时时钟 |
| calc | calc(a){op}(b){=c}{cmd} | 执行算术运算并可选执行命令 |
| if | if(cond){cmd} else(cond){cmd} | 条件分支执行 |
| url | url | 显示网络功能开发状态 |
| connect | connect | 显示 HarRead 阅读站 URL |

### 变量系统

#### 命名规则

- 首字符必须是字母或下划线
- 后续字符可以是字母、数字或下划线
- 区分大小写
- 最大长度 15 字符

#### 示例

```
set counter = 0
set max_value = 100
set result = a * b + c
```

### 表达式语法

| 优先级 | 运算符 | 说明 |
|--------|--------|------|
| 1 | () | 括号分组 |
| 2 | * / | 乘法、除法 |
| 3 | + - | 加法、减法 |

### 条件语句

```
if(condition){ command }
if(condition){ command } else{ command }
```

### 算术运算

```
calc(10){+}(12){= c}{say result}
calc(a){*}(b){= result}{show result}
```
