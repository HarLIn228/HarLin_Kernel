## 内核概述

HarLin Kernel 是由 HarLin228 Studio 开发的 x86 通用操作系统内核。本项目采用统一 API 层设计，第三方开发者可基于本内核修改实现、添加新功能并在遵守 MIT 开源协议的前提下发布衍生作品。

当前版本已移除内置 Shell，启动后直接进入静默等待状态，所有扩展功能由第三方开发者通过 API 实现。

## 内存布局

| 地址范围 | 用途 | 说明 |
|----------|------|------|
| 0x00000000 - 0x000003FF | 实模式中断向量表 | 保留 |
| 0x00000400 - 0x000004FF | BIOS 数据区 | 保留 |
| 0x00000500 - 0x00007BFF | 可用常规内存 | 约 30KB |
| 0x00007C00 - 0x00007DFF | 引导扇区 | 512 字节 |
| 0x00010000 - 0x0009FFFF | 内核加载区 | 约 520KB |
| 0x000A0000 - 0x000B7FFF | 图形帧缓冲 | VGA 图形模式 |
| 0x000B8000 - 0x000BFFFF | VGA 文本缓冲区 | 80x25 字符显示 |

## 统一 API 接口

所有功能通过 harlin_API.h 中声明的接口访问，入口函数为 Harlin_Boot()。

### 控制台输出

```c
void Harlin_ConClear(void);
void Harlin_ConPutChar(char c);
void Harlin_ConPrint(const char* str);
void Harlin_ConPrintHex(u32 val);
void Harlin_ConPrintDec(s32 val);
void Harlin_ConSetColor(u8 fg, u8 bg);
```

### 显示模式

```c
#define HARLIN_DISP_VGA_TEXT 0
#define HARLIN_DISP_VGA_13H  1
#define HARLIN_DISP_VESA     2

int  Harlin_DisplaySetMode(int mode);
void Harlin_DisplayClear(unsigned char color);
void Harlin_DisplayPutPixel(int x, int y, unsigned char color);
void Harlin_DisplayPutString(int x, int y, const char* str, unsigned char color);
```

当前内置驱动仅支持 HARLIN_DISP_VGA_TEXT。HARLIN_DISP_VGA_13H 与 HARLIN_DISP_VESA 为第三方扩展点，需在 display.c 中接入对应驱动实现。

### 键盘输入

```c
char Harlin_KeyGet(void);
```

## 第三方开发指南

### 添加显示驱动

在 src/Sys_C/display.c 的 display_set_mode 函数中接入新驱动：

```c
if (mode == HARLIN_DISP_VGA_13H) {
    vga13_init();   // 第三方实现
    current_mode = mode;
    return 0;
}
```

并相应实现 display_clear / display_put_pixel / display_put_string 的分支逻辑。

### 添加硬件驱动

在 src/Sys_C/ 目录下创建新驱动源文件，在 harlin_API.c 的 Harlin_Boot() 中初始化，并通过 harlin_API.h 暴露接口。

### 注意事项

- 内核运行于 32 位保护模式，无虚拟内存与内存分页
- 无堆内存管理，建议使用静态分配
- 标准 C 库不可用，使用内核提供的工具函数
- 所有 API 区分大小写
- 第三方扩展需遵守 MIT 开源协议
